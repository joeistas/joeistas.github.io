CalcTab Function Descriptions

Numpad Keypad

  • Digits 0-9 and .
    • Use these buttons to enter numbers
  • (-)
    • Used to make the following number a negative number
  • ÷
    • Binary operator for division. Sometimes shown as "/"
  • ×
    • Binary operator for multiplication. Shown as "·" in the Results History and Equation Bar
  • -
    • Binary operator for subtraction.
  • +
    • Binary operator for addition.

Math Keypad

  • Exp
    • Multiplies the previous number by 10 to the power following the operator. Form: (operand)E(power)
  • log
    • Logarithmic function. Defaults to a base of 10. Form: log{base}(operand) base is optional
  • logx
    • Shortcut to change the base of the log function. See log
  • e
    • Mathematical constant e
  • eⁿ
    • Mathematical constant e to a power. Form: eⁿ(power)
  • ln
    • Natural Logarithm. Form: ln(operand)
  • √
    • Root function. Defaults to square root. Form: √{root}(operand)
  • ⁿ√
    • Shortcut to change the root of the Root Function. See Root Function
  • xⁿ
    • Power Function. Multiply the operand by itself the number of times in the power. Form: (operand)^(power)
  • (
    • Parenthesis, used to group operators and operands.
  • {
    • Brackets, used to change options on some functions.

Trig Keypad

  • sin
    • Trig function sine
  • cos
    • Trig function cosine
  • tan
    • Trig function tangent
  • asin
    • Inverse sine
  • acos
    • Inverse cosine
  • atan
    • Inverse tangent
  • sinh
    • Hyperbolic sine
  • cosh
    • Hyperbolic cosine
  • tanh
    • Hyperbolic tangent
  • π
    • Mathematical constant π
