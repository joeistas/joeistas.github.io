Slice Keyboard Version 2.1

Jun 25, 2013

Slice Keyboard version 2.1 is available for the full and demo versions. Version 2.1 fixes an issue that would cause unintentional gestures on some devices that register touch events when a finger was hovering over the screen, most notably the Nexus 7.
