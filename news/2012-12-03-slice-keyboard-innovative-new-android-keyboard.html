Slice Keyboard: Touch typing on a touch screen

Dec 2, 2012

The Slice Keyboard is an innovative touch keyboard design that makes it possible to touch type on a touch screen. Slice's main innovation is a blank home row where you place your fingers. Then keys are shown depending on which fingers are placed on the home row. Each key is in a direction relative to the keys on the home row. This allows you to type without having to look at the keys. The home row positions can statically placed or adjust to your finger placement. Also you have easy access to punctuation, special characters and a number pad. Slice keyboard supports Android tablets 7" and larger.

Slice Keyboard comes in two versions, the full version and a demo version.

The full version can be used throughout the Android operating system and with all Android apps that work with the system keyboard. The full version of Slice Keyboard also comes with a typing area, and a Space Invaders like game for typing practice. Purchase the full version of Slice Keyboard here.

Slice Keyboard Demo only comes with the typing area, and a Space Invaders like game for typing practice. The demo version can not be used to replace the system keyboard. Slice Keyboard Demo can be downloaded in the Google Play Store here.
