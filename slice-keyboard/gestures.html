Slice Keyboard Gestures

Slice Keyboard supports several gestures to make typing faster and more fluid.

Capitalization Gesture

Capitalize any key by pressing and flinging on any key.

Space Gesture

Add a space by pressing a Home Key and flinging to the Right.

Delete Gesture

Delete a character by pressing a Home Key and flinging to the Left

Hand Switch Gesture

When typing with one hand switch the keys to opposite hand's keys by pressing a Home Key and flinging Up. The Swap Hand Gesture will work only when typing with one hand.

Display Preview Mode

Preview Mode displays all of the keys at once. To open Preview Mode swipe in from the left or right edges of the screen.
