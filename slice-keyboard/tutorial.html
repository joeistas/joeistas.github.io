Slice Keyboard Tutorial

Slice Keyboard is designed to make touch typing on a touch screen possible. The keys are placed in specific directions relative to each other to aid typing without looking down at the keys. Since the layout of the keyboard can change based on different screen sizes and orientations.

Home Keys

The circular keys in the middle of the keyboard are the Home Keys. Pressing the Home Keys will display the Slice keys. The layout of Slice Keyboard can changed based on the size of the device and it's current orientation.

Typing

Each Home Key has a set of keys that are displayed when it is not pressed but at least one other Home Key is pressed. The keys are in directions relative to it's Home Key
